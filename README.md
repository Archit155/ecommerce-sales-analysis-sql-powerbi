# E-Commerce Sales Analysis (SQL & Power BI)

This project focuses on analyzing an e-commerce dataset to uncover key business insights related to sales performance, customer behavior, and operational efficiency. The analysis was performed using SQL for data manipulation and Power BI for visualization.

---

## 🎯 Objectives

- Analyze overall sales and revenue trends
- Identify top-performing and loss-making products
- Understand customer distribution across cities
- Evaluate payment method preferences
- Measure delivery performance and efficiency

---

## 🧠 Exploratory Data Analysis (EDA)

- Cleaned and joined multiple tables (orders, customers, products, payments) using SQL
- Analyzed revenue trends over time
- Identified loss-making products
- Explored city-wise sales distribution
- Evaluated delivery time patterns

---

## 🛠️ Tools & Technologies

- **SQL** - Data Cleaning & Transformation
- **Power BI** - Dashboard & Visualization
- **DAX** - Calculated Measures

---

## 📊 Key Metrics (KPIs)

- Total Revenue
- Total Orders
- Average Order Value
- Estimated Profit Margin
- Average Delivery Days

---

## 📈 Dashboard Features

- Revenue vs Profit Trend Analysis
- Top 10 Loss-Making Products
- Top Revenue-Generating Cities
- Payment Method Distribution
- Delivery Performance Analysis

---

## 🔍 Key Insights

- Revenue shows a consistent upward trend over time
- Credit card is the most preferred payment method
- A few products contribute significantly to losses
- Sales are concentrated in top cities
- Most deliveries are completed within a limited time range

---

## 📝 Note

**Profit Calculation:**
```
Profit = Payment Value – Freight Value
```

This is an approximation due to the absence of product cost data.
